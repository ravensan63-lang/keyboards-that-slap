# Keyboards that slap

The best mechanical keyboards at every price point.

## Budget (under $80)
- [Keychron K2](https://keychron.com) - Wireless, compact, great value
- [Epomaker TH80](https://epomaker.com) - Hot-swap, gasket mount

## Mid-range ($80–200)
- [NuPhy Air75](https://nuphy.com) - Ultra thin, wireless
- [Mode Envoy](https://modedesigns.com) - Premium feel, polycarbonate

## Endgame ($200+)
- [HHKB Professional](https://hhkeyboard.us) - Topre switches, legendary feel
- [Keycult No. 2](https://keycult.com) - Custom, limited run
